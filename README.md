# Course Hunter client
A web client for [course_hunter](https://github.com/dashMrl/course_hunter) .

## Quick Start 
Switch the request url to your, then exec:
```shell
$ npm i
$ npm start
```

## Publish 
```shell
$ npm i
$ npm run build
``` 
> It's recommended to separate you client and server
